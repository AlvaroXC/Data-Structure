PASOS PARA COMPILAR EL PROGRAMA:
1.Descomprimir el archivo  zip
2. Iniciar la línea de comandos y abrir la ubicación del proyecto(carpeta descomprimida)
3.Para compilar el programa, ingresar en la línea de comandos: javac <Carpeta de una de las tres listas>/*.java
Ejemplo: javac Link/*.java
4. Para ejecutar el programa, ingresar en la línea de comandos: java <archivo main>
Ejemplo: java LinkListApp
5. Después de haber seguido los pasos, el programa empezará a ejecutarse
